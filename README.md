# Lifehacker OS

A lifelong personal operating system. Local-first, AI-native, extensible.

> This system exists to help its owner understand life over decades.

## Structure

```
.claude/       ← AI brain (Claude's understanding of you)
docs/          ← Human & AI readable documentation
apps/          ← Applications (desktop, future: mobile, web)
packages/      ← Reusable modules (core, storage, ai, plugins)
```

## Principles

- Local First — all critical data lives locally
- Data Longevity — data survives framework changes
- Modular — remove or add features safely
- AI-Native — AI assists cognition, doesn't replace agency
